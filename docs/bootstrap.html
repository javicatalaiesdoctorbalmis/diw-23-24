Bootstrap

  • Mobile-first, responsive frontend (CSS + JavaScript) toolkit, like Tailwind or Bulma.
  • Free and open-source (one of the most popular projects on GitHub, +160.000 stars).
  • Includes prebuilt componentes (buttons, navbars,...) and a grid layout system.
  • Easy to customize (with CSS variables and Sass).

Using Bootstrap

  • Using CDN links (CSS and JavaScript) in HTML.
  • Download ready-to-use compiled code.
  • Download source files (Sass and JavaScript).
  • Add source code using a package manager (npm)

Reboot

  • Bootstrap overrides the browser's default styles for many HTML elements.
  • Reboot provides an elegant, consistent, and simple baseline to build upon.
  • It only uses element selectors (like p, h1-h6 or ul/ol)

Utility classes

  • Margin and padding:
    • The classes are named using the format {property}{sides}-{size}.
    • property is m for margin or p for padding.
    • sides is one of: t (top), b (bottom), s (start), e (end), x (start and end), y (top and bottom) or blank (all 4 sides).
    • size is a number between 0 and 5.

Layout

  • Bootstrap's layout system is responsive and mobile-first.
  • Bootstrap includes six default breakpoints:
    • Extra extra large (xxl): >= 1400px
    • Extra large (xl): >= 1200px
    • Large (lg): >= 992px
    • Medium (md): >= 768px
    • Small (sm): >= 576px
    • Extra small (no infix): < 576px

Layout

  • Containers are the most basic layout element in Bootstrap.
  • Bootstrap comes with three different containers:
    • .container, which sets a max-width at each responsive breakpoint.
    • .container-{breakpoint}, which is width: 100% until the specified breakpoint.
    • .container-fluid, which is width: 100% at all breakpoints.
  • Containers are required when using Bootstrap default grid system

Layout

  • Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content.
  • It's built with flexbox and is fully responsive.
  • There are 12 template columns per row, allowing you to create different combinations of elements that span any number of columns.
  • You can control container and column sizing and behavior by each breakpoint.

Customizing Bootstrap

  • There are multiple ways to customize Bootstrap:
    • Add or override Bootstrap’s styles.
    • Using CSS variables.
    • Using Bootstrap's Sass source files.
  • There are two ways to get Bootstrap's source files:
    • Download the source files from GitHub.
    • Install Bootstrap's source files using npm.
    •                             
npm install bootstrap

Customizing Bootstrap

  • Whenever possible, avoid modifying Bootstrap’s core files.
  • Create your own stylesheet (main.scss, custom.scss) that imports Bootstrap.
  • We have two options:
    • Include all of Bootstrap.
    • Include only the parts you need.

Customizing Bootstrap

  • Including all of Bootstrap:
                            
// Include any default variable overrides here 
// Functions won't be available!
$primary: #c29938;

// Include all of Bootstrap
@import "../node_modules/bootstrap/scss/bootstrap";

// Then add additional custom code here
...

Customizing Bootstrap

  • Including only the parts you need:
                            
// Include functions first
@import "../node_modules/bootstrap/scss/functions";

// Include any default variable overrides here 
$primary: #c29938;

// Include remainder of required Bootstrap stylesheets
@import "../node_modules/bootstrap/scss/variables";

// Include any default map overrides here
$custom-colors: (
  "custom-color": #900
);
$theme-colors: map-merge($theme-colors, $custom-colors);


// Include remainder of required parts
@import "../node_modules/bootstrap/scss/maps";
@import "../node_modules/bootstrap/scss/mixins";
@import "../node_modules/bootstrap/scss/root";

// Optionally include any other parts as needed
@import "../node_modules/bootstrap/scss/utilities";
@import "../node_modules/bootstrap/scss/reboot";
@import "../node_modules/bootstrap/scss/type";
@import "../node_modules/bootstrap/scss/images";
@import "../node_modules/bootstrap/scss/containers";
@import "../node_modules/bootstrap/scss/grid";

// Add additional custom code here
